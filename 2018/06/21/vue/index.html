4. bus.js实现兄弟之间组件通信（官方文档中只提供了父与子通信）

在文件夹中新建一个文件，随意取名，例如bus.js放在了assets文件夹（文件夹自定义）下，在bus.js文件夹下写入：

import Vue from 'vue'
export default new Vue()

在两个要通信的兄弟组件之间引入bus.js

import bus from '你bus.js文件存放的位置'

想要传递信息的兄弟1

<button @click="toBus"></button>

methods: {
    toBus () {
        bus.$emit('on', '这个是兄弟1传给兄弟2的值')
    }
}

接受传递信息的兄弟2

mounted () {
    bus.$on('on', (msg) => {
        console.log(msg) // msg就是从兄弟1接到的数据
    })
}
